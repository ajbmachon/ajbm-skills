### US-006: Implement trace list command
Status: in_progress
Depends on: US-004

Description:
As a user, I want to list recent traces so that I can see what's happening in my AI application.

Acceptance Criteria:
- [ ] 'trace list' fetches recent traces with optional filters
- [ ] Supports filters: --limit, --name, --user-id, --session-id
- [ ] Uses v2 Observations API with cursor-based pagination
- [ ] Output shows: trace ID, name, timestamp, status (success/error)
- [ ] Progress indicator for fetches >5 seconds
- [ ] Example: 'trace list --limit 10' -> shows 10 most recent traces
- [ ] Example: 'trace list --name chatbot' -> filters by trace name
- [ ] Negative case: No traces found -> 'No traces found matching your criteria'
- [ ] ISC reference: rows 14, 19
