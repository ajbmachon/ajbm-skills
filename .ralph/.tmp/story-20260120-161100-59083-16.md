### US-011: Implement experiment subcommand
Status: in_progress
Depends on: US-004

Description:
As a user, I want experiment commands so that I can create datasets and run A/B tests on my prompts.

Acceptance Criteria:
- [ ] 'experiment create-dataset' creates a new dataset
- [ ] 'experiment add-item' adds an item to a dataset
- [ ] 'experiment run' runs experiment using run_experiment() API
- [ ] 'experiment compare' compares results of experiment runs
- [ ] Uses concurrent execution (run_experiment() handles this)
- [ ] Example: 'experiment create-dataset --name test-cases --description "Golden test cases"'
- [ ] Example: 'experiment run --dataset test-cases --model gpt-4o'
- [ ] Negative case: Dataset not found -> 'Dataset "[name]" not found. Use experiment create-dataset first.'
- [ ] ISC reference: rows 29-34
