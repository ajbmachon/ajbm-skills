# Kimi Research Findings — February 2026

## Kimi K2 (Foundation)
- **Released:** July 2025
- **Architecture:** 1T MoE, 32B active per token
- **Context:** 128K-256K tokens
- **Training data:** 15.5T tokens
- **License:** Modified MIT
- **Design:** Agentic-first — perceive environments, call tools, execute multi-step plans
- **Temperature:** 0.6 (official recommendation)
- **Tool calling:** Nearly 100% accuracy via official API
- **Built-in tools:** 10+ ready-to-use (web search, etc.)
- **Prompting:** Goal-oriented, not step-by-step

## Kimi K2.5 (Latest — January/February 2026)
- **Released:** Late January 2026, open weights February 2026
- **Architecture:** 1T MoE, 32B active (same base as K2)
- **Key addition:** Native multimodal — text, images, video, documents
- **Training:** ~15T mixed visual and text tokens
- **License:** MIT (fully open-weight)
- **Pricing:** $0.60/M input, $3/M output
- **Four modes:** Instant, Thinking, Agent, Agent Swarm (100 sub-agents, 1500+ parallel tool calls)
- **Benchmarks:** Comparable to GPT-5.2, Claude Opus 4.6, Gemini 3 Pro
